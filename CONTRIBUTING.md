## Contributing to AxonOS

Thanks for your interest in improving AxonOS.

### Quick Start

- Fork the repo and create a feature branch.
- Keep PRs small and focused.
- Prefer adding or updating documentation when behavior changes.

### Pull Requests

- Describe **what** changed and **why**.
- Include testing notes (what you ran, what you verified).
- Avoid committing generated artifacts (build outputs, large binaries, runtime data).

### Security Issues

If your change relates to a potential security issue, please follow `SECURITY.md` and avoid posting sensitive details publicly.
